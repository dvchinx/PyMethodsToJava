<h2> Actual Methods </h2>

**print():**

- print(str, op)     
str arg, op  -> 
"NORMAL", "DOWN", "INVERSE", "INVERSE-DOWN"

- print(integer) --> int integer

- print(arrStr) --> String[] arrStr

- print(arrInt) --> int[] arrInt

- print(arrFloat) --> float[] arrFloat
